# Assistente-de-Bordo
Criação de um app que funciona como um assistente de bordo, onde ficara registrado o momento correto de fazer manutenções e serviços no seu veiculo.


A ideia de criação desse app se deu olhando para uma necessidade pessoal de não perder a hora de fazer as manutenções e entender melhor a "saúde" do meu veículo, além de ser uma experiência de uso do react native e do node,js

A ideia é simples:  Um app minimalista onde o usuário cadastra a quilometragem atual e o serviço realizado. O app cruza isso com os manuais básicos e cria um semáforo visual (Verde, Amarelo, Vermelho) para os próximos desgastes.

O Diferencial: Sem menus complexos de oficinas. Apenas um botão de "Atualizar Hodômetro" e um checklist do que precisa de atenção.
